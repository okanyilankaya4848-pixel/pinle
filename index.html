📱 Yeni cihaz algılandı. Firebase bilgilerini bir kez daha gir — bundan sonra bu cihazda tekrar sorulmaz.
1
Firebase Console'a git ve giriş yap
console.firebase.google.com adresini aç.
Sağ üstten Google hesabınla giriş yap.
2
Yeni proje oluştur
+ Proje ekle butonuna tıkla.
Proje adı gir (örn: pinle-app) → İleri → İleri → Proje oluştur.
3
Firestore veritabanını aç
Sol menüde Oluştur > Firestore Database ya da
Build > Firestore Database seç.
→ Veritabanı oluştur tıkla → Test modunda başlat seç → Etkinleştir.
4
Web uygulaması ekle
Sol üstteki ⚙️ Project Settings (dişli çark) sekmesine tıkla.
Sayfayı aşağı kaydır → Uygulamalarınız bölümünde
</> Web ikonuna tıkla.
Uygulama adı gir (örn: pinle-web) → Uygulamayı kaydet.
5
Config değerlerini kopyala
Karşına çıkan kodda şu 3 değeri bul ve aşağıdaki alanlara yapıştır:
apiKey: "AIzaSy..."
projectId: "pinle-xxxxx"
appId: "1:123456:web:abc..."
EmailJS Kurulumu (3 adım, 5 dakika):
1. emailjs.com'a git → ücretsiz kayıt ol
2. Email Services → Gmail'i bağla → Service ID'yi kopyala
3. Email Templates → Şablon oluştur → Template ID'yi kopyala
4. Account → Public Key'i kopyala

☁️ Firebase Ayarları
Config değerlerini bulmak için:

① console.firebase.google.com → Projeyi seç
② Sol üstteki ⚙️ Project Settings dişli çarkına tıkla
③ Genel sekmesinde aşağı in → Uygulamalarınız bölümüne gel
④ Web uygulaması yoksa </> ikonuna tıkla → kaydet
⑤ Firebase SDK snippet → Config seçeneğini seç
→ apiKey, projectId, appId değerlerini kopyala
